  • • Combine cores: Use .bg-success com .text-white para melhor contraste

💡 Exemplos de Uso Completo

Diferentes formas de usar o Yor CSS em seus projetos

1. Uso Mínimo (apenas tokens + botões):

<link rel="stylesheet" href="packages/yor-tokens/dist/yor-tokens.css">
<link rel="stylesheet" href="packages/yor-buttons/dist/yor-buttons.css">
<link rel="stylesheet" href="themes/material/dist/material-theme.css">

2. Uso Completo (todos os módulos):

<!-- Base -->
<link rel="stylesheet" href="packages/yor-tokens/dist/yor-tokens.css">


<link rel="stylesheet" href="packages/yor-buttons/dist/yor-buttons.css">
<link rel="stylesheet" href="packages/yor-cards/dist/yor-cards.css">
<link rel="stylesheet" href="packages/yor-containers/dist/yor-containers.css">
<link rel="stylesheet" href="packages/yor-spacing/dist/yor-spacing.css">
<link rel="stylesheet" href="packages/yor-typography/dist/yor-typography.css">
<link rel="stylesheet" href="packages/yor-colors/dist/yor-colors.css">


<link rel="stylesheet" href="themes/fluent/dist/fluent-theme.css">

3. Uso Seletivo (apenas o que precisa):

<!-- Para um projeto que só usa cards e tipografia -->
<link rel="stylesheet" href="packages/yor-tokens/dist/yor-tokens.css">
<link rel="stylesheet" href="packages/yor-cards/dist/yor-cards.css">
<link rel="stylesheet" href="packages/yor-typography/dist/yor-typography.css">
<link rel="stylesheet" href="themes/cupertino/dist/cupertino-theme.css">

4. Troca dinâmica de temas:

<link rel="stylesheet" href="themes/material/dist/material-theme.css" id="theme-css">


function changeTheme(themeName) {
  const themeLink = document.getElementById('theme-css');
  themeLink.href = `themes/${themeName}/dist/${themeName}-theme.css`;
}

✅ Melhores Práticas

Dicas para usar o Yor CSS de forma eficiente

📋 Ordem de importação:

  1. <strong>1. Tokens</strong> - Sempre primeiro (obrigatório)
  2. <strong>2. Componentes</strong> - Na ordem que preferir
  3. <strong>3. Tema</strong> - Sempre por último
  4. <strong>4. CSS customizado</strong> - Após o tema (se necessário)

🎯 Performance:

✅ Faça

  • • Use apenas módulos necessários
  • • Combine arquivos CSS em produção
  • • Use CDN para arquivos estáticos
  • • Minifique os arquivos CSS

❌ Evite

  • • Importar todos os módulos se não usar
  • • Sobrescrever muitas variáveis CSS
  • • Misturar múltiplos temas
  • • Ignorar a ordem de importação

🔧 Customização:

Para customizar cores ou espaçamentos:

/* Seu CSS customizado - após o tema */
:root {
  --color-primary: #your-color !important;
  --space-custom: 20px;
}

/* Ou crie variantes específicas */
.btn--custom {
  background-color: var(--color-primary);
  border-radius: var(--radius-lg);
}

Instruções para desenvolvedores que querem modificar o Yor CSS

Comandos disponíveis:

                  npm run build

                  # Build apenas packages
                  npm run build:packages

                  # Build apenas temas
                  npm run build:themes

                  # Build específico
                  npm run build:tokens
                  npm run build:buttons
                  npm run build:cards
