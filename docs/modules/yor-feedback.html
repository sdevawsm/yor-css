Yor Feedback

Introdução

O módulo yor-feedback fornece componentes para feedback visual ao usuário, como alertas, barras de progresso, spinners, tooltips e modais.

Dica: Todos os componentes de feedback são totalmente responsivos e seguem as melhores práticas de acessibilidade.

Instalação

Para instalar o módulo yor-feedback, você pode usar npm:

npm install @yor-css/feedback

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/feedback.css">

Componentes

Alertas

Alertas são usados para exibir mensagens importantes ao usuário. Podem ter diferentes variantes e suportam ícones e ações.

Alertas
✓
Sucesso!
Operação realizada com sucesso.
!
Erro!
Ocorreu um erro ao processar sua solicitação.
!
Atenção!
Esta ação não pode ser desfeita.
i
Informação
Sistema em manutenção programada.
<!-- Alerta de Sucesso -->
<div class="alert alert--success">
    <div class="alert__icon">✓</div>
    <div class="alert__content">
        <div class="alert__title">Sucesso!</div>
        <div class="alert__message">Operação realizada com sucesso.</div>
    </div>
    <button class="alert__close">×</button>
</div>

<!-- Alerta de Erro -->
<div class="alert alert--error">
    <div class="alert__icon">!</div>
    <div class="alert__content">
        <div class="alert__title">Erro!</div>
        <div class="alert__message">Ocorreu um erro ao processar sua solicitação.</div>
    </div>
    <button class="alert__close">×</button>
</div>

<!-- Alerta de Aviso -->
<div class="alert alert--warning">
    <div class="alert__icon">!</div>
    <div class="alert__content">
        <div class="alert__title">Atenção!</div>
        <div class="alert__message">Esta ação não pode ser desfeita.</div>
    </div>
    <button class="alert__close">×</button>
</div>

<!-- Alerta de Informação -->
<div class="alert alert--info">
    <div class="alert__icon">i</div>
    <div class="alert__content">
        <div class="alert__title">Informação</div>
        <div class="alert__message">Sistema em manutenção programada.</div>
    </div>
    <button class="alert__close">×</button>
</div>

Barras de Progresso

Barras de progresso são usadas para mostrar o progresso de uma operação.

Barras de Progresso
<!-- Barra de Progresso Básica -->
<div class="progress">
    <div class="progress__bar" style="width: 75%"></div>
</div>

<!-- Barra de Progresso Listrada -->
<div class="progress progress--striped">
    <div class="progress__bar" style="width: 50%"></div>
</div>

<!-- Barra de Progresso Animada -->
<div class="progress progress--animated">
    <div class="progress__bar"></div>
</div>

Tooltips

Tooltips são usados para mostrar informações adicionais.

Tooltips
Tooltip básico
Tooltip abaixo
Tooltip à esquerda
Tooltip à direita
<!-- Tooltip Básico -->
<div class="tooltip">
    <button class="btn">Hover me</button>
    <div class="tooltip__content">Tooltip básico</div>
</div>

<!-- Tooltip Abaixo -->
<div class="tooltip" data-tooltip-position="bottom">
    <button class="btn">Hover me</button>
    <div class="tooltip__content">Tooltip abaixo</div>
</div>

<!-- Tooltip à Esquerda -->
<div class="tooltip" data-tooltip-position="left">
    <button class="btn">Hover me</button>
    <div class="tooltip__content">Tooltip à esquerda</div>
</div>

<!-- Tooltip à Direita -->
<div class="tooltip" data-tooltip-position="right">
    <button class="btn">Hover me</button>
    <div class="tooltip__content">Tooltip à direita</div>
</div>

Animações

O módulo yor-feedback inclui animações suaves para todos os componentes:

  • Alertas com fade in/out
  • Progress bars com transições suaves
  • Spinners com rotação contínua
  • Tooltips com fade in/out
  • Modais com slide in/out

Acessibilidade

Todos os componentes de feedback seguem as melhores práticas de acessibilidade:

  • ARIA roles e labels
  • Foco visível
  • Contraste adequado
  • Suporte a leitores de tela
  • Navegação por teclado
