Forms

Componentes de formulário para melhorar a experiência de entrada de dados

Visão Geral

O módulo Forms fornece componentes estilizados e acessíveis para criar formulários modernos e responsivos. Todos os componentes são customizáveis através de variáveis SCSS e seguem as melhores práticas de acessibilidade.

Exemplo Básico

<form class="yor-form">
  <div class="yor-form-group">
    <label class="yor-form-label" for="name">Nome</label>
    <input type="text" class="yor-input" id="name" placeholder="Digite seu nome">
  </div>
  <div class="yor-form-group">
    <label class="yor-form-label" for="email">Email</label>
    <input type="email" class="yor-input" id="email" placeholder="Digite seu email">
  </div>
  <div class="yor-form-group">
    <label class="yor-form-label" for="message">Mensagem</label>
    <textarea class="yor-textarea" id="message" placeholder="Digite sua mensagem"></textarea>
  </div>
  <button type="submit" class="yor-btn yor-btn-primary">Enviar</button>
</form>
