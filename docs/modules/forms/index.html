Forms

Componentes de formulário para melhorar a experiência de entrada de dados

O módulo Forms fornece componentes estilizados e acessíveis para criar formulários modernos e responsivos. Todos os componentes são customizáveis através de variáveis CSS e seguem as melhores práticas de acessibilidade.

Visão Geral

O módulo Forms inclui componentes para todos os tipos de entrada de dados, desde campos de texto simples até seletores complexos, com suporte a validação, estados de erro e diferentes tamanhos. 

<form class="yor-form">
  <div class="yor-form-group">
    <label class="yor-form-label" for="name">Nome</label>
    <input type="text" class="yor-input" id="name" placeholder="Digite seu nome">
  </div>
  <div class="yor-form-group">
    <label class="yor-form-label" for="email">Email</label>
    <input type="email" class="yor-input" id="email" placeholder="Digite seu email">
  </div>
  <div class="yor-form-group">
    <label class="yor-form-label" for="message">Mensagem</label>
    <textarea class="yor-textarea" id="message" placeholder="Digite sua mensagem"></textarea>
  </div>
  <button type="submit" class="btn btn--primary">Enviar</button>
</form>

Temas

O módulo Forms suporta diferentes temas e design systems. Você pode alternar entre eles usando os atributos data-theme e data-theme-system.

Tema Claro/Escuro

<!-- Adicione o atributo data-theme ao elemento html ou body -->
<html data-theme="light">
  <form class="yor-form">
    <div class="yor-form-group">
      <label class="yor-form-label" for="theme-name">Nome</label>
      <input type="text" class="yor-input" id="theme-name" placeholder="Digite seu nome">
    </div>
    <!-- ... -->
  </form>
</html>

<script>
function toggleTheme() {
  const html = document.documentElement;
  const currentTheme = html.getAttribute('data-theme');
  html.setAttribute('data-theme', currentTheme === 'dark' ? 'light' : 'dark');
}
</script>

Design Systems

<!-- Adicione o atributo data-theme-system ao elemento html ou body -->
<html data-theme-system="default">
  <form class="yor-form">
    <div class="yor-form-group">
      <label class="yor-form-label" for="system-name">Nome</label>
      <input type="text" class="yor-input" id="system-name" placeholder="Digite seu nome">
    </div>
    <!-- ... -->
  </form>
</html>

<script>
function changeThemeSystem(system) {
  document.documentElement.setAttribute('data-theme-system', system);
}
</script>
