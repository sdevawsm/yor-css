Yor Navigation

Introdução

O módulo yor-navigation fornece componentes de navegação estilizados e acessíveis para estruturar a interface do usuário de forma intuitiva e responsiva.

Dica: Todos os componentes de navegação são totalmente responsivos e seguem as melhores práticas de acessibilidade.

Instalação

Para instalar o módulo yor-navigation, você pode usar npm:

npm install @yor-css/navigation

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/navigation.css">

Responsividade

O módulo yor-navigation fornece comportamentos responsivos para todos os componentes:

  • Navbar colapsa em menu hamburguer em telas menores
  • Tabs se adaptam ao tamanho da tela
  • Breadcrumbs podem ser truncados em telas pequenas
  • Paginação se ajusta ao espaço disponível
  • Dropdowns se posicionam automaticamente
Dica: Use as classes de breakpoint para personalizar o comportamento responsivo dos componentes.

Acessibilidade

Todos os componentes de navegação seguem as melhores práticas de acessibilidade:

  • Navegação por teclado
  • ARIA roles e labels
  • Foco visível
  • Contraste adequado
  • Estrutura semântica
Nota: Sempre forneça textos alternativos para ícones e elementos visuais.
