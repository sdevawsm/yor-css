Yor CSS

Um framework CSS modular e moderno para desenvolvimento web.

O Yor CSS é um framework CSS modular que permite que você use apenas os componentes que precisa, mantendo seu projeto leve e eficiente. Cada módulo é independente e pode ser usado separadamente.

Características

  • Modular - Use apenas o que você precisa
  • Leve - Sem dependências desnecessárias
  • Moderno - Suporte a recursos CSS modernos
  • Responsivo - Design adaptável a qualquer dispositivo
  • Customizável - Fácil de personalizar e estender

Começando

Para começar a usar o Yor CSS, você pode instalar os módulos via npm: 

npm install @yor-css/forms @yor-css/buttons @yor-css/cards

Ou incluir os arquivos CSS diretamente via CDN: 

<link rel="stylesheet" href="https://cdn.yor-css.com/forms.css">
<link rel="stylesheet" href="https://cdn.yor-css.com/buttons.css">
<link rel="stylesheet" href="https://cdn.yor-css.com/cards.css">
