Começando

Para começar a usar o Yor CSS, você pode instalar os módulos via npm: 

npm install @yor-css/forms @yor-css/buttons @yor-css/cards

Ou incluir os arquivos CSS diretamente via CDN: 

<link rel="stylesheet" href="https://cdn.yor-css.com/forms.css">
<link rel="stylesheet" href="https://cdn.yor-css.com/buttons.css">
<link rel="stylesheet" href="https://cdn.yor-css.com/cards.css">
