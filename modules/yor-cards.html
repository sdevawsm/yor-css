Yor Cards

Introdução

O módulo yor-cards fornece componentes de cards estilizados e flexíveis para exibição de conteúdo.

Dica: Os cards são totalmente responsivos e podem ser facilmente customizados para diferentes necessidades.

Instalação

Para instalar o módulo yor-cards, você pode usar npm:

npm install @yor-css/cards

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/cards.css">

Componentes

Cards Básicos

Cards são usados para agrupar e exibir conteúdo relacionado.

Card Básico

Título do Card

Conteúdo do card aqui. Este é um exemplo de card básico com título e corpo.

<div class="card">
    <div class="card__header">
        <h3 class="card__title">Título do Card</h3>
    </div>
    <div class="card__body">
        <p>Conteúdo do card aqui. Este é um exemplo de card básico com título e corpo.</p>
    </div>
    <div class="card__footer">
        <button class="btn btn--primary">Ação</button>
    </div>
</div>

Variantes

Cards podem ter diferentes estilos e variantes para diferentes contextos.

Cards com Imagem

Card com Imagem

Este card inclui uma imagem no topo.

<div class="card">
    <img src="imagem.jpg" alt="Imagem do card" class="card__image">
    <div class="card__body">
        <h3 class="card__title">Card com Imagem</h3>
        <p>Este card inclui uma imagem no topo.</p>
    </div>
</div>

Animações

O módulo yor-cards inclui animações suaves para melhorar a experiência do usuário:

  • Hover com elevação suave
  • Transições suaves para cards clicáveis
  • Animações de entrada para grid de cards
  • Efeitos de hover para cards interativos

Acessibilidade

Todos os cards seguem as melhores práticas de acessibilidade:

  • Estrutura semântica com roles apropriados
  • Contraste adequado para texto e imagens
  • Suporte a navegação por teclado
  • Textos alternativos para imagens
  • Foco visível para cards interativos
