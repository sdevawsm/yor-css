Yor Spacing

Introdução

O módulo yor-spacing fornece um sistema consistente de espaçamento para criar layouts harmoniosos e responsivos.

Dica: O sistema de espaçamento usa uma escala baseada em múltiplos de 4px para manter a consistência visual.

Instalação

Para instalar o módulo yor-spacing, você pode usar npm:

npm install @yor-css/spacing

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/spacing.css">

Acessibilidade

O sistema de espaçamento segue as melhores práticas de acessibilidade:

  • Espaçamento adequado para leitura
  • Suporte a zoom e redimensionamento
  • Compatibilidade com leitores de tela
  • Ordem de leitura lógica
  • Contraste adequado entre elementos
