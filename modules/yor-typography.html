Yor Typography

Introdução

O módulo yor-typography fornece um sistema completo de tipografia para criar interfaces consistentes e legíveis.

Dica: O sistema de tipografia é baseado em uma escala modular que mantém a harmonia visual em toda a interface.

Instalação

Para instalar o módulo yor-typography, você pode usar npm:

npm install @yor-css/typography

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/typography.css">

Acessibilidade

A tipografia segue as melhores práticas de acessibilidade:

  • Hierarquia clara de headings
  • Contraste adequado entre texto e fundo
  • Tamanhos de fonte legíveis
  • Espaçamento adequado entre linhas
  • Suporte a zoom e redimensionamento
