Yor Container

Introdução

O módulo yor-container fornece um sistema flexível de containers para estruturar o layout de suas páginas.

Dica: Os containers são responsivos e se adaptam automaticamente a diferentes tamanhos de tela.

Instalação

Para instalar o módulo yor-container, você pode usar npm:

npm install @yor-css/container

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/container.css">

Containers

Containers Básicos

O container básico centraliza o conteúdo e define uma largura máxima.

Container Padrão

Este é um container padrão com largura máxima e centralizado.

<div class="container">
    <p>Este é um container padrão com largura máxima e centralizado.</p>
</div>

Containers Fluid

O container fluid ocupa 100% da largura disponível.

Container Fluid

Este é um container fluid que ocupa toda a largura disponível.

<div class="container container--fluid">
    <p>Este é um container fluid que ocupa toda a largura disponível.</p>
</div>

Containers Responsivos

Containers podem ter diferentes larguras máximas em diferentes breakpoints.

Container Responsivo

Este container se adapta a diferentes tamanhos de tela.

<div class="container container--responsive">
    <p>Este container se adapta a diferentes tamanhos de tela.</p>
</div>
/* Breakpoints do container responsivo */
.container--responsive {
    max-width: 100%;
    padding: 0 var(--space-4);
}

@media (min-width: 640px) {
    .container--responsive {
        max-width: 640px;
    }
}

@media (min-width: 768px) {
    .container--responsive {
        max-width: 768px;
    }
}

@media (min-width: 1024px) {
    .container--responsive {
        max-width: 1024px;
    }
}

@media (min-width: 1280px) {
    .container--responsive {
        max-width: 1280px;
    }
}

Acessibilidade

Os containers seguem as melhores práticas de acessibilidade:

  • Estrutura semântica
  • Suporte a navegação por teclado
  • Compatibilidade com leitores de tela
  • Ordem de leitura lógica
  • Suporte a zoom e redimensionamento
