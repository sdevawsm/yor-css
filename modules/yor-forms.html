Yor Forms

Introdução

O módulo yor-forms fornece componentes de formulário estilizados e acessíveis para melhorar a experiência de entrada de dados em suas aplicações.

Dica: Todos os componentes de formulário são totalmente responsivos e seguem as melhores práticas de acessibilidade.

Instalação

Para instalar o módulo yor-forms, você pode usar npm:

npm install @yor-css/forms

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/forms.css">

Componentes

Inputs

Inputs são campos de texto básicos que podem ser usados para coletar dados do usuário.

Input Básico
<div class="form-group">
    <label class="form-group__label">Nome</label>
    <input class="input" type="text" placeholder="Digite seu nome">
</div>
Nota: Sempre use o componente form-group para envolver seus inputs, pois ele fornece o espaçamento e alinhamento corretos.

Checkboxes

Checkboxes permitem que os usuários selecionem uma ou mais opções de uma lista.

Checkbox Básico
<label class="checkbox">
    <input type="checkbox" class="checkbox__input">
    <span class="checkbox__box"></span>
    <span class="checkbox__label">Aceito os termos</span>
</label>

Radio Buttons

Radio buttons permitem que os usuários selecionem uma única opção de um conjunto de opções mutuamente exclusivas.

Radio Button Básico
<label class="radio">
    <input type="radio" name="option" class="radio__input">
    <span class="radio__box"></span>
    <span class="radio__label">Opção 1</span>
</label>

Selects

Selects permitem que os usuários escolham uma opção de uma lista suspensa.

Select Básico
<div class="form-group">
    <label class="form-group__label">País</label>
    <div class="select">
        <select class="select__native">
            <option value="">Selecione um país</option>
            <option value="br">Brasil</option>
            <option value="pt">Portugal</option>
            <option value="es">Espanha</option>
        </select>
    </div>
</div>

Textareas

Textareas são campos de texto multilinha para entrada de texto mais longa.

Textarea Básico
<div class="form-group">
    <label class="form-group__label">Mensagem</label>
    <textarea class="textarea" placeholder="Digite sua mensagem..."></textarea>
</div>

Form Groups

Form groups são containers que agrupam labels, inputs e mensagens de ajuda/erro.

Form Group Completo
Digite um email válido 
<div class="form-group">
    <label class="form-group__label form-group__label--required">Email</label>
    <input class="input" type="email" placeholder="seu@email.com">
    <span class="form-group__help">Digite um email válido</span>
</div>

Validação

O módulo yor-forms fornece classes e estilos para validação de formulários.

Input com Erro
Digite um email válido 
<div class="form-group">
    <label class="form-group__label">Email</label>
    <input class="input is-invalid" type="email" placeholder="seu@email.com">
    <span class="form-group__error">Digite um email válido</span>
</div>

Acessibilidade

Todos os componentes de formulário seguem as melhores práticas de acessibilidade:

  • Labels associados corretamente aos inputs
  • Estados de foco visíveis
  • Mensagens de erro claras e associadas aos campos
  • Suporte a navegação por teclado
  • Contraste adequado
Dica: Sempre use labels descritivos e forneça mensagens de erro claras para melhorar a acessibilidade.
