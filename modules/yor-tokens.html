Yor Tokens

Introdução

O módulo yor-tokens fornece um sistema completo de tokens de design para criar interfaces consistentes e acessíveis.

Dica: Os tokens são variáveis CSS que podem ser facilmente customizadas para diferentes temas e sistemas de design.

Instalação

Para instalar o módulo yor-tokens, você pode usar npm:

npm install @yor-css/tokens

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/tokens.css">

Cores

Cores Primárias

As cores primárias são usadas para elementos principais da interface.

Cores Primárias
--color-primary
Cor Primária
--color-primary-variant
Variação Primária

Acessibilidade

Os tokens de design seguem as melhores práticas de acessibilidade:

  • Contraste adequado entre cores
  • Tamanhos de fonte legíveis
  • Espaçamento consistente
  • Suporte a zoom
  • Compatibilidade com leitores de tela
