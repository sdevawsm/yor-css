Yor Colors

Introdução

O módulo yor-colors fornece um sistema de cores consistente e flexível para seus projetos.

Dica: O sistema de cores é totalmente customizável e suporta temas claro e escuro.

Instalação

Para instalar o módulo yor-colors, você pode usar npm:

npm install @yor-css/colors

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/colors.css">

Temas

O sistema de cores suporta diferentes temas e sistemas de design:

  • Material Design
  • Fluent Design
  • Cupertino
  • Tema Claro
  • Tema Escuro
/* Aplicar tema */
:root {
    /* Tema claro */
    --color-background: #ffffff;
    --color-surface: #f5f5f5;
    --color-text-body: #333333;
}

/* Tema escuro */
[data-theme="dark"] {
    --color-background: #121212;
    --color-surface: #1e1e1e;
    --color-text-body: #ffffff;
}

Acessibilidade

O sistema de cores segue as diretrizes de acessibilidade:

  • Contraste mínimo de 4.5:1 para texto normal
  • Contraste mínimo de 3:1 para texto grande
  • Suporte a modo de alto contraste
  • Cores não são o único meio de transmitir informação
  • Testes de daltonismo
