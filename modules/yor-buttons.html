Yor Buttons

Introdução

O módulo yor-buttons fornece componentes de botões estilizados e acessíveis para sua aplicação.

Dica: Todos os botões são totalmente responsivos e seguem as melhores práticas de acessibilidade.

Instalação

Para instalar o módulo yor-buttons, você pode usar npm:

npm install @yor-css/buttons

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/buttons.css">

Componentes

Botões Básicos

Botões são usados para ações e interações do usuário. Podem ter diferentes estilos e variantes.

Botões Básicos
<button class="btn">Botão Padrão</button>
<button class="btn btn--primary">Botão Primário</button>
<button class="btn btn--secondary">Botão Secundário</button>
<button class="btn btn--outline">Botão Outline</button>
<button class="btn btn--ghost">Botão Ghost</button>

Variantes

Botões podem ter diferentes variantes de cor para diferentes contextos.

Variantes de Cor
<button class="btn btn--primary">Primário</button>
<button class="btn btn--secondary">Secundário</button>
<button class="btn btn--outline">Outline</button>
<button class="btn btn--ghost">Ghost</button>

Tamanhos

Botões podem ter diferentes tamanhos para diferentes contextos.

Tamanhos
<button class="btn btn--sm">Pequeno</button>
<button class="btn">Médio</button>
<button class="btn btn--lg">Grande</button>
<button class="btn btn--xl">Extra Grande</button>

Estados e Formas

Botões podem ter diferentes estados e formas para feedback visual.

Estados
<button class="btn" disabled>Desabilitado</button>
<button class="btn btn--primary btn--block">Botão Bloco</button>
Formas
<button class="btn btn--rounded">Botão Arredondado</button>
<button class="btn btn--square">Botão Quadrado</button>
<button class="btn btn--pill">Botão Pill</button>

Animações

O módulo yor-buttons inclui animações suaves para todos os estados:

  • Hover com fade suave
  • Active com scale
  • Loading com rotação
  • Transições suaves entre estados

Acessibilidade

Todos os botões seguem as melhores práticas de acessibilidade:

  • ARIA roles e labels
  • Foco visível
  • Contraste adequado
  • Suporte a leitores de tela
  • Navegação por teclado
