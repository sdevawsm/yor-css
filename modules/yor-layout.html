Yor Layout

Introdução

O módulo yor-layout fornece um conjunto completo de utilitários para criar layouts flexíveis e responsivos.

Dica: Os utilitários de layout são projetados para trabalhar em conjunto com o sistema de grid e flexbox.

Instalação

Para instalar o módulo yor-layout, você pode usar npm:

npm install @yor-css/layout

Ou incluir via CDN:

<link rel="stylesheet" href="https://cdn.yor-css.com/layout.css">

Acessibilidade

Os utilitários de layout seguem as melhores práticas de acessibilidade:

  • Estrutura semântica
  • Ordem de leitura lógica
  • Suporte a zoom e redimensionamento
  • Compatibilidade com leitores de tela
  • Navegação por teclado
